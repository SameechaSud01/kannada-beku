Kannada Beku — Manual Testing Plan

A complete, screen-by-screen manual QA pass for the Expo learning app.

Covers onboarding, auth, lessons, all four games, guide, emergency, heritage, progress & sync, audio, offline behaviour, and cross-cutting UI/accessibility checks.
Drafted 2026-07-02 · branch feature/lesson-illustrations · Expo SDK 54
How to use this plan. Work top to bottom on a fresh install, then repeat the device-matrix checks (Suite P) on each target device. Each test has a stable ID (e.g. AUTH-03) — record Pass / Fail / Blocked and, on failure, a note + screenshot. Priorities: Critical must pass before release; High ship-blocking for the affected area; Medium polish/edge.
Pass = matches expected Fail = deviates Blocked = can't test

Prerequisites

  • Devices (foundation target matrix): iPhone SE 2nd/3rd gen (375×667, the small-screen floor), iPhone 15 Pro Max, and a mid-size Android. iOS tablet (iPad) for supportsTablet.
  • Builds: test on a dev/preview build, not only Expo Go, so Supabase auth, OAuth redirects, deep links, and Sentry behave as in production.
  • Accounts: at least one fresh email/password account, one Google account, one Apple ID (iOS), plus one existing account with prior progress (for the rebrand/streak-adoption checks).
  • State reset: know how to fully reset — delete & reinstall the app (clears AsyncStorage: user_prefs, sync_queue, progress) and, for server data, reset the test user's rows in Supabase.
  • Network control: be able to toggle Airplane mode / Wi-Fi off mid-flow to exercise the offline sync queue.
  • Screenshots: iOS xcrun simctl io booted screenshot /tmp/screen.png; Android adb exec-out screencap -p > /tmp/screen.png.
Reset between major suites where noted — several tests (first-run, unlock gating, streak) only reproduce on a clean state.

✦Regression Smoke (run every build)

  • ONB-06 — no onboarding re-loop after completion.
  • AUTH-02 / AUTH-07 — login works, session persists.
  • LRN-03 / GAM-02 — completing content unlocks the next lesson / game part.
  • HOME-03 / HOME-07 — rings update after activity; Home and Profile agree.
  • LES-04 / AUD-01 — audio plays.
  • CONV-02 — conversation content fully scrollable.
  • OFF-01 / OFF-02 — offline completion, then flush on reconnect.
  • STK-02 — same-day activity doesn't double the streak.
  • UI-01 / UI-05 — iPhone SE layout + safe areas.
